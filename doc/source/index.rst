pySecDec
========

`pySecDec` [PSD17]_ is a toolbox for the calculation of dimensionally regulated parameter integrals
using the sector decomposition approach [BH00]_; see also [Hei08]_, [BHJ+15]_.

Please cite the following references if you use `pySecDec` for a scientific publication:
 * pySecDec [PSD17]_
 * CUBA [Hah05]_, [Hah16]_
 * FORM [Ver00]_, [KUV13]_, [RUV17]_
 * GSL [GSL]_
 * nauty [BKAP]_ (if you use `dreadnaut`) 
 * normaliz [BIR]_, [BIS16]_ (if you use a geometric decomposition strategy)
